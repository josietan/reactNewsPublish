#React_NewsSystem
##技术栈
	前端框架：React + React-router + Redux;<br>
	界面：bootstrap + jquery;<br>
	编译工具：webpack +babel + ES2016;<br>
	后端接口：PHP + MySQL;<br>
##项目启动
	该项目需要MySQL数据库支持
##项目简介
